# 🍽️ Catalogue des Restos Tunisie

Application de catalogue de restaurants avec commande directe via WhatsApp.

## 📱 Fonctionnalités

- 🍕 Affichage des restaurants et menus
- 🔍 Recherche de plats et restaurants
- 🛒 Panier d'achat (max 5 articles)
- 📞 Commande directe sur WhatsApp
- ⏰ Vérification des horaires d'ouverture

## 🚀 Utilisation

1. Ouvrez `index.html` dans votre navigateur
2. Parcourez les restaurants et les plats
3. Ajoutez au panier ou commandez directement
4. Confirmez la commande via WhatsApp

## 🛡️ Sécurité

- ✅ Aucune donnée personnelle n'est stockée
- ✅ Aucune base de données
- ✅ Aucune authentification
- ✅ Toutes les commandes sont envoyées directement sur WhatsApp
- ✅ Application front-end pure (HTML/CSS/JS)
